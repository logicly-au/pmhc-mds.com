#### PMHC MDS Update v3.1.0 ####

We are pleased to announce that PMHC MDS Update v3.1.0 was released on 
March 20, 2026.

The following new reports have been implemented:
* A13: Data quality report — Volume of completed outcome measures report

The following features have been applied in the reporting interface:
* Added ANZAC day (replacement) public holiday for NSW for reports that 
  count business days

The following features have been applied in the user management interface:
* Users with the User Management role can now see inactive organisations 
  and remove roles that users have at inactive organisations. Roles 
  cannot be added for a user at an inactive organisation.

The user guides have been updated to reflect these updates. They are available
at [https://docs.pmhc-mds.com/](https://docs.pmhc-mds.com/){:target="_blank"}.
