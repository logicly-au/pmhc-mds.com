---
title: Final reminder - End of life for PMHC MDS v4.1 and Way Back v3.0 specifications
---

The current version of the core PMHC MDS data specification is Version 5.0. 
This has been available for upload since the 18th of December 2025.

Since the 18th of December 2026 there has been a transition period allowing 
for uploads in PMHC MDS Version 4.1 and The Way Back Version 3.0 extension 
formats. This transition period is intended to give organisations time to 
understand and implement the change.

After the 30th of June 2026 the PMHC MDS will no longer accept uploads using 
the Version 4.1 or The Way Back Version 3.0 Extension formats. If you are 
able to upload in Version 5.0 now, please start using this format immediately.

*Note: There is no requirement imposed by the system to wait until the 30th 
of June 2026 before supplying in Version 5.0 format.*

Please contact support@pmhc-mds.com if you will not be able to upload in the 
PMHC MDS Version 5.0 format by 30th of June 2026.

PMHC MDS data specifications are documented at [https://docs.pmhc-mds.com/data-specifications.html#current-versions](https://docs.pmhc-mds.com/data-specifications.html#current-versions){:target="_blank"}

A summary of the changes between version 4.1 and 5.0 is available at [https://docs.pmhc-mds.com/projects/data-specification/en/v5.0/upgrading-from-previous-versions.html](https://docs.pmhc-mds.com/projects/data-specification/en/v5.0/upgrading-from-previous-versions.html){:target="_blank"}.
