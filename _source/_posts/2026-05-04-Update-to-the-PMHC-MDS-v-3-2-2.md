#### PMHC MDS Update v3.2.2 ####

We are pleased to announce that PMHC MDS Update v3.2.2 was released on 
May 4, 2026.

As per the Department of Health, Disability and Ageing’s (DoHDA)
Circular on 30 April, 2026 which is published at
[https://pmhc-mds.com/communications/#/2026/04/30/FOR-ACTION-New-MMHC-system-tags-in-the-PMHC-MDS/](https://pmhc-mds.com/communications/#/2026/04/30/FOR-ACTION-New-MMHC-system-tags-in-the-PMHC-MDS/){:target="_blank"}
the tags !rhcpsychology and !rhcpsychiatry are now available through Data Entry,
Upload and Data Extracts in the PMHC MDS.

Version 5.0.3 of the base PMHC MDS data specification has been released to
add additional validations to support these tags.

Please refer to DoHDA’s Circular linked above for details about usage of these tags.


The following updates have been applied in the data entry interface:
* Fixed a bug in the add client form that could stop navigation away from the
  unsaved data modal.

The following updates have been applied in the reporting interface:
* Updated the table headers in the A16 report to clarify what each column is counting.

The following updates have been applied in the data extract interface:
* Fixed a bug in table pagination where the number of rows displayed would not change
  to more than 10.


The user guides have been updated to reflect these updates. They are available
at [https://docs.pmhc-mds.com/](https://docs.pmhc-mds.com/){:target="_blank"}.
