PMHC MDS Update v0.6.1 was released on February 10, 2017.

The following functionality has been added:

  1. Sub tabs have been added to each of the ‘Organisations’, ‘Users’ and ‘Upload’ tabs in order to group information in a more user friendly manner.
  2. Uploads can now include a value of 9 - Missing/Not stated for Episode - Homelessness

The user guide has been updated to reflect these updates. It is available at https://docs.pmhc-mds.com/user-documentation/

PMHC MDS Data Specifications Update v1.0.4 was released on February 10, 2017.

The updates includes the following changes:

  1. Updated the data model diagram to explicitly show Primary Health Network Organisations and Provider Organisations
  2. Added more description to Organisation Path for Client, Episode, Service Contact, Collection Occasion and Practitioner records
  3. Episode - Homelessness Flag updated to include missing/not stated value of 9

The Department of Health’s ‘Department response to issues raised by PHNs’ document has now been incorporated into the online Frequently Asked Questions section at https://docs.pmhc-mds.com/faqs/concepts-processes/. Answers to questions will be added there. The existing PDF will no longer be maintained and will be removed from the website.
